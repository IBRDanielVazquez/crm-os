# 🏹 Amazon Email Marketing Systems Architect

Este repositorio contiene la arquitectura estratégica y técnica para construir un activo de Email Marketing profesional para marcas en Amazon.

## 🌟 Misión del Sistema
Transformar ventas volátiles en Amazon en relaciones propias, automatizadas y rentables mediante una infraestructura externa controlada.

## 📂 Estructura del Proyecto
- **01_strategy**: Posicionamiento, funnels y cumplimiento (Amazon SES).
- **02_audience**: Segmentación avanzada y ciclo de vida del cliente.
- **03_copy**: Secuencias maestras (Welcome, Launch, Win-back).
- **04_automation**: Triggers, webhooks y lógica de eventos.
- **05_infrastructure**: Configuración de AWS SES, DNS y reputación.
- **06_tracking**: KPIs, Dashboards y atribución.
- **07_sops**: Manuales operativos para ejecución repetible.
- **08_prompts**: Habilidades para Antigravity y asistentes IA.
- **09_assets**: Lead magnets y creatividades.
